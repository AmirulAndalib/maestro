# Dependencies

The kernel needs some external components in order to work, here is the list.



## Mandatory

### Compilation

- The *Nightly* Rust toolchain



### Booting

- A bootloader supporting *Multiboot2* (example: `GRUB2`)



## Optional

### Debugging

- `GDB`: GNU Debugger
- `QEMU`: Emulator
- `Bochs`: Emulator
- `Virtualbox`: Emulator



### Documentation

- `mdbook`: Documentation generator
